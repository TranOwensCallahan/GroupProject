Course Calender Project
Jenna Tran, Evan Owens, Austin Callahan
using impress.js framework
Overview of Course Calender
  • A WPF Application
  • Modeled similarly to our class course calender

Why a course calender?

  • Thought it would be useful beyond this classroom
  • Concept/Idea really interested us

How does it work?

  • An excel file is created
  • User inputs topic name, number of days the topic will be covered, topic notes, and holiday dates
  • User chooses the first day of the semester
  • Course calender is created
